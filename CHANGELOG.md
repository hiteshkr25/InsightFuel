## [1.0.0] - 2026-07-14

### Added
*   **GitOps Workflows:** Integrated ArgoCD Application manifests for progressive deployments.
*   **External Secrets Operator:** Enabled credentials synchronization from Vault backend directly to Kubernetes pods.
*   **Validation Suite:** Added Python-based post-deployment smoke test runners.
*   **Aesthetics & Observability:** Implemented RED/USE metrics Grafana dashboard layouts.
*   **Product Health Engine:** Core analytics engines evaluating metrics.
*   **AI Recommendations Engine:** Reusable client modules generating recommendations.
*   **Power BI Embedded:** API routes generating secure Entra RLS tokens.
