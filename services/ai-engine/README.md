# InsightFuel AI Recommendation Engine

The AI Recommendation Engine aggregates performance, adoption, growth and retention signals to generate actionable, explainable recommendations.

## Run Commands

1.  **Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Start API server:**
    ```bash
    uvicorn app.main:app --reload --port 3007
    ```

3.  **Run tests:**
    ```bash
    python -m pytest
    ```
